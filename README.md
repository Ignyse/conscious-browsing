# conscious-browsing
## Idea
Main project idea is to have a list of urls that the user want to limit time spent on it.
For example, entertainment website which one opens by habits, even for no inital reason, and then stays on it because of the dopamine.
This tool would launch a video/image sent by the user connected to the url in question that will make the user question whether he really wants to open it or not. It is kind of like more "conscious" browsing, or in other words guilt-tripping yourself, to stop wasting time on something not worth it.

## Stack
Originally it will be built as a web extension (firefox initially). Mainly hmtl, javascript, css + list of url linked to uploaded videos
